jps：查看当前用户下的java进程

常用参数

参数 说明
-q 只打印进程id
-m 打印传入main方法的参数
-l 打印完整包名、应用主类名、jar的完整路径
-v 打印传递给jvm的参数

常用命令

查看当前用户下的java进程详情

jps -lvm

jinfo：查看java进程运行时的jvm参数

常用参数

参数 说明
-flag name 打印参数name的值
-flag [+|-]name 开启关闭参数name
-flag name=value 设定参数name的值为value
-flags 打印全部flag参数
-sysprops 打印系统属性

常用命令

查看全部flag参数

jinfo -flags <pid>

查看java运行时参数

  • -XX:+PrintFlagsInitial 打印参数初始值
  • -XX:+PrintFlagsFinal 打印参数最终值
  • -XX:+PrintCommandLineFlags 打印命令行参数
  • -XX:+UnlockExperimentalVMOptions 解锁实验性参数
  • -XX:+UnlockDiagnosticVMOptions 解锁诊断参数

查看可管理的java运行时参数最终值

java -XX:+PrintFlagsFinal -version | grep manageable

jmap：查看java进程的内存情况

常用参数

参数 说明
-heap 打印堆内存详情
-histo[:live] 打印堆内存所有的对象实例和对象大小，live只打印存活对象，为保证打印信息可靠性，该操作会暂停应用，须谨慎使用
-dump:[live,]format=b,file=filename.hprof 导出堆内存所有对象实例到文件，live只导出存活对象，为保证导出信息可靠性，该操作会暂停应用，须谨慎使用
-finalizerinfo 输出在等待执行finalize方法的对象
-clstats 打印java类加载器的智能统计信息

特别注明：histo:live和dump:live的过滤存活对象操作，先触发一次GC行为，再进行对应的操作，须谨慎使用。

常用命令

# 打印堆内存详情
jmap -heap <pid>

# 导出堆内存所有对象实例和对象大小
jmap -dump:format=b,file=filename.hprof <pid>

jstat：查看jvm运行状态信息

常用参数

参数 说明
-gc gc信息
-gcutil gc信息统计
-gccause 最后一次gc和原因
-gccapacity 堆内存各区大小
-gcnew 新生代信息统计
-gcnewcapacity 新生代大小
-gcold 老年代信息统计
-gcoldcapacity 老年代大小
-class 加载class数量及占用空间

常用命令

jstat -gc <pid> [<interval> [<count>]]
